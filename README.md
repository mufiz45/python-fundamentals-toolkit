# Python Fundamentals Toolkit

## Overview
A collection of command-line applications built to demonstrate core Python programming concepts including functions, conditional logic, validation, and modular structure.

## Projects Included
- Personal Data Collector (BMI calculator included)
- Calculator Application
- Number Analyzer
- Password Strength Validator

## Skills Demonstrated
- Input handling
- Functions
- Control flow
- Error handling
- Modular programming

## How to Run
Clone repository and run:
python filename.py
